# JavaScript and CSS Clock
This is a simple clock project built with HTML, CSS, and JavaScript. It displays an analog clock with hour, minute, and second hands, indicating the current time.

## Features
- **Analog Clock**: The clock is displayed in an analog format, with hour, minute, and second hands.
- **Real-time-update**: The clock is updated in real-time, with smooth transitions between time updates.
- **Smooth Animation:** Smooth animation is applied to the movement of the clock hands.
- **Responsive Design**: The clock is designed to be fully responsive and works on all screen sizes.

## Installation

1. Clone the repository:
```
git clone https://github.com/Omowunmikamil/JavaScript-30
```
2. Navigate to the project directory
3. __Open the `index.html` file in your browser.__

## Usage
- The clock will automatically display the current time.
- Watch the clock hands move in real-time to indicate the current time.
